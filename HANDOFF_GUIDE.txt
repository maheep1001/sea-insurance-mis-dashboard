COMMON BUG SCENARIOS TO WATCH FOR:
─────────────────────────────────────────────────────────────────────────────

1. B1/B2 Race Condition
   - Multiple concurrent getActuals() calls write to NAV summary B1/B2
   - Fix: Ensure loadCountry() calls are sequential (await each before next)
   - Status: Already implemented; verify no regression

2. QTD Edge Cases
   - Single-month selection (monthIdx=0) should sum all to QTD
   - Closing month boundary (29/30/31) varies by month
   - Status: Logic is in buildQTDDescriptor() — test thoroughly

3. Cache Invalidation
   - Switching date range should invalidate old cache key
   - Freezing range should not affect new range's cache
   - Status: cacheBustAll() exists; verify it covers all keys

4. FY26 Q4 Lookup
   - Q1 previous quarter comes from FY26 Q4 (Jan-Mar 26)
   - This data is NOT in live NAV — only in Insurance Actuals
   - Status: getActualsFromFile() handles this; test with Q1 load

5. Frozen State Per Tab
   - PH and INDO should track freeze state independently
   - localStorage keys include country name
   - Status: Implementation complete; test tab-switching
